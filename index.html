üìö Quick Reference: Three Main Software Types

üîß System Software

Purpose: Controls hardware and provides platform for other software
Examples:
  • Operating Systems: Windows, macOS, Linux, Android, iOS
  • Device Drivers: Graphics drivers, printer drivers
  • System Utilities: Antivirus, disk cleanup, system monitors
  • Firmware: BIOS/UEFI, router firmware

üì± Application Software

Purpose: Performs specific tasks for end users
Examples:
  • Productivity: Microsoft Office, Google Workspace
  • Creative: Photoshop, Premiere Pro, Blender
  • Communication: Zoom, WhatsApp, Gmail
  • Entertainment: Netflix, Spotify, Steam games
  • Business: Salesforce, QuickBooks, Slack

‚öôÔ∏è Programming Software

Purpose: Tools for creating, debugging, and maintaining other software
Examples:
  • IDEs: Visual Studio, Eclipse, VS Code
  • Text Editors: Sublime Text, Atom, Notepad++
  • Compilers/Interpreters: GCC, Python interpreter
  • Version Control: Git, GitHub, GitLab
  • Databases: MySQL, PostgreSQL, MongoDB
