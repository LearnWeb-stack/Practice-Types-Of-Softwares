üîß System Software

Purpose: Controls hardware and provides platform for other software

Operating Systems
Windows 11, macOS Sonoma, Ubuntu Linux, Android, iOS, Chrome OS
Device Drivers
Graphics card drivers (NVIDIA, AMD), printer drivers, audio drivers, network adapters
System Utilities
Antivirus (Norton, McAfee), disk cleanup tools, registry cleaners, system monitors, backup software
Firmware
BIOS/UEFI, router firmware, smartphone bootloaders, embedded system software
System Libraries
DirectX, OpenGL, system APIs, runtime environments (.NET Framework, Java Runtime)

üì± Application Software

Purpose: Performs specific tasks for end users

Productivity Software
Microsoft Office, Google Workspace, LibreOffice, Notion, Evernote
Creative & Multimedia
Adobe Photoshop, Premiere Pro, Blender, Canva, GIMP, Audacity
Communication
Zoom, Microsoft Teams, WhatsApp, Discord, Slack, Gmail, Outlook
Entertainment
Netflix, Spotify, Steam, Epic Games, VLC Media Player, YouTube
Business Applications
Salesforce, QuickBooks, SAP, Oracle ERP, Shopify, HubSpot
Educational
Khan Academy, Duolingo, Coursera, Blackboard, Moodle
Web Browsers
Chrome, Firefox, Safari, Edge, Opera
